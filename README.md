# Winstarchitect Skills Repository

Het complete skill-ecosysteem voor Winstarchitect. 7 skills, 38 reference files, 1 geïntegreerd acquisitiesysteem.

## Architectuur

```
shared-kernel (fundament)
    ↓ voedt alle skills
    ├── goated-ads (advertenties)
    ├── core-story-webinar (strategie-engine)
    │       ↓ Blueprint voedt:
    │       ├── funnel-pages (pagina's)
    │       ├── funnel-scripts (video scripts)
    │       └── goated-ads (hooks uit Blueprint)
    ├── analytics-feedback (data → optimalisatie)
    └── winst-os (technisch platform)
```

## Orchestratie Protocol

```
Stap 0: Universal Client Brief (1x per klant)
Stap 1: ToV Kalibratie (1x per klant)
Stap 2: core-story-webinar → Perceptie Blueprint
Stap 3 (parallel): funnel-pages + goated-ads + funnel-scripts
Stap 4: Quality Gate op ALLE output
Stap 5: Launch → Analytics → Iteratie (doorlopend)
```
