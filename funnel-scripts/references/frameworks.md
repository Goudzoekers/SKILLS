# Frameworks — Script Structuren per Type

8 script-types met hun structuur. Laden bij elk script.

## 1. Pre-Event VSL (1-2 min)

Doel: urgentie creëren om te registreren/komen.

```
[HOOK:] Confronterende vraag of statement (5 sec)
[PROBLEEM:] Herkenbaar scenario — "als je X herkent..." (15 sec)
[BELOFTE:] Wat ze leren in het event (15 sec)
[BEWIJS:] 1 kort resultaat of autoriteit (10 sec)
[CTA:] Exacte instructie — "klik hieronder en vul je naam in" (10 sec)
[URGENTIE:] Waarom NU (5 sec)
```

## 2. Thank You Page Video (2-4 min)

Doel: excitement opbouwen, verwachtingen zetten, show-up verhogen.

```
[BEDANKT:] Persoonlijke bevestiging (15 sec)
[WAT ER GAAT GEBEUREN:] 3 concrete dingen die ze leren (30 sec)
[WAAROM DIT ANDERS IS:] Onderscheid van andere trainingen (30 sec)
[VERWACHTINGEN:] Wat ze moeten doen voor het event (30 sec)
[TEASE:] Eén specifiek inzicht dat je deelt (30 sec)
[INSTRUCTIE:] Agenda, check inbox, eventueel survey invullen (30 sec)
```

## 3. Upsell Video (3-6 min)

Doel: upsell verkopen direct na registratie.

```
[HOOK:] "Voordat je verdergaat — ik heb iets speciaals" (10 sec)
[CONTEXT:] Waarom dit relevant is nu ze geregistreerd zijn (30 sec)
[AANBOD:] Wat ze krijgen — concreet (60 sec)
[BEWIJS:] Resultaat van iemand die de upsell gebruikte (30 sec)
[BEZWAAR:] Prijs-neutralisatie (30 sec)
[CTA:] Klik op de knop hieronder (15 sec)
[ALTERNATIEF:] "Als het niet voor jou is — geen probleem, je plek is al bevestigd" (10 sec)
```

## 4. Sales VSL — Hoofdaanbod (8-20 min)

Doel: hoofdaanbod verkopen. Kan standalone of post-webinar.

```
[HOOK:] Confronterend + specifiek (30 sec)
[PROBLEEM:] 3 herkenbare scenario's — in hun woorden (2 min)
[VIJAND:] Waarom het probleem blijft — het is niet hun schuld (2 min)
[SHIFT:] Het onverwachte inzicht (2 min)
[OPLOSSING:] Introductie methode/programma (1 min)
[WAT JE KRIJGT:] Module voor module — met zonder/zodat (3 min)
[BEWIJS:] 3 case studies (3 min)
[PRIJS:] Waarde stack → investering (1 min)
[GARANTIE:] Risico wegnemen (30 sec)
[BEZWAREN:] Top 3 — benoem en neutraliseer (2 min)
[CROSSROADS:] Optie A vs Optie B (1 min)
[CTA:] Exacte instructie + wat er gebeurt na de klik (30 sec)
```

## 5. Recap Video (5-15 min)

Doel: webinar replay samenvatten voor afwezigen.

```
[HOOK:] "Je hebt de masterclass gemist — hier zijn de key takeaways" (15 sec)
[SAMENVATTING:] 3-5 kernpunten uit het webinar (5-8 min)
[BEWIJS:] 1-2 reacties van aanwezigen (1-2 min)
[AANBOD:] Kort — verwijzing naar sales page (1-2 min)
[CTA:] "Bekijk de volledige replay" of "Ga naar de sales page" (15 sec)
```

## 6. Onboarding Video (2-5 min)

Doel: nieuwe klant activeren, eerste stappen duidelijk maken.

```
[WELKOM:] Persoonlijk, warm, bevestigend (30 sec)
[WAT JE HEBT:] Overzicht van wat ze gekocht hebben (30 sec)
[EERSTE STAP:] Exact wat ze NU moeten doen (1 min)
[VERWACHTINGEN:] Tijdlijn, wat ze kunnen verwachten (30 sec)
[SUPPORT:] Hoe ze hulp krijgen (30 sec)
[MOTIVATIE:] Afsluiting met energie + identiteit shift (30 sec)
```

## 7. Registratie Video (1-2 min)

Doel: op een registratiepagina — conversie verhogen.

```
[HOOK:] Direct aanspreken + probleem (10 sec)
[BELOFTE:] Wat ze leren — 3 punten (20 sec)
[AUTORITEIT:] 1-2 credibility punten (15 sec)
[CTA:] Vul je naam in hieronder (10 sec)
[URGENTIE:] Beperkt / datum / exclusief (5 sec)
```

## 8. Launch Video (3-8 min)

Doel: nieuw product/programma aankondigen.

```
[HOOK:] Grote aankondiging — energie (15 sec)
[CONTEXT:] Waarom dit bestaat — het verhaal erachter (1-2 min)
[WAT HET IS:] Naam + kernbelofte (30 sec)
[VOOR WIE:] Ideale klant beschrijven (30 sec)
[WAT JE KRIJGT:] Overzicht (1-2 min)
[BEWIJS:] Beta-resultaten of eigen resultaat (1 min)
[AANBOD:] Prijs + launch-bonus (30 sec)
[CTA:] Instructie + urgentie (15 sec)
```

## Universele Script-Regels

1. Schrijf voor het OOR, niet het oog — korte zinnen, geen bijzinnen
2. [PAUZE] markeren waar de spreker even stopt
3. [NADRUK] markeren bij kernwoorden
4. Max 150 woorden per minuut spreektijd
5. Elke sectie = 1 emotie (niet mixen)
6. CTA = letterlijke instructie, niet suggestie
