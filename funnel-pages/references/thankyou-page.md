# Thank You Page — Bevestigingspagina

Na registratie of aankoop. Doel: bevestigen, verwachtingen zetten, volgende stap activeren.

## Variant A: Na Gratis Registratie

```
[HEADLINE:] Top! Je staat op de lijst.
[SUBHEADLINE:] Check je inbox — je bevestigingsmail is onderweg.

Wat er nu gebeurt:
1. ✅ Je ontvangt een bevestigingsmail (check ook spam)
2. 📅 Zet [DATUM + TIJD] in je agenda [LINK .ics]
3. 📱 24 uur voor de masterclass krijg je een herinnering

[OPTIONEEL: SURVEY — zie core-story-webinar/references/thank-you-page-survey.md]
[OPTIONEEL: UPSELL — zie upsell-page.md]
```

## Variant B: Na Aankoop

```
[HEADLINE:] Welkom! Je bent binnen.
[SUBHEADLINE:] Dit is de beste beslissing die je dit jaar hebt genomen.

Wat er nu gebeurt:
1. ✅ Je ontvangt een bevestigingsmail met inloggegevens
2. 🎥 Start hier: [LINK naar eerste module/onboarding]
3. 💬 Join de community: [LINK]
4. 📞 Je eerste sessie plannen: [LINK scheduler]

[TESTIMONIAL:] "[Quote van recente klant over hun eerste week]"
```

## Variant C: Na Lead Magnet Download

```
[HEADLINE:] Je [lead magnet] is onderweg!
[SUBHEADLINE:] Check je inbox — je ontvangt het binnen 2 minuten.

Terwijl je wacht:
[VIDEO — 2-3 min: bedank + tease volgende stap]
— of —
[TEKST:] Hier zijn 3 dingen die je direct kunt doen met [lead magnet]:
1. [Actie 1]
2. [Actie 2]
3. [Actie 3]
```

## Kernprincipes

- Bevestig DIRECT wat ze krijgen en wanneer
- Geef exacte volgende stappen (1-2-3 format)
- Verminder buyer's remorse met social proof of bevestiging
- Activeer de volgende stap in de funnel (survey, upsell, onboarding)
- Geen navigatiebalk — houd ze op de pagina
