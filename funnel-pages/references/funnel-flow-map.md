# Funnel Flow Map — Configuraties & Routing

Visueel overzicht van alle funnel-types en hun pagina-volgorde.

## Configuratie A: Webinar Funnel (primair)

```
[AD] → [REGISTRATIE PAGE] → [UPSELL] → [THANK YOU + SURVEY]
                                  ↓ nee
                            [DOWNSELL] → [THANK YOU + SURVEY]
                                  ↓ nee
                            [THANK YOU + SURVEY]

[PRE-WEBINAR MAILS x11] → [WEBINAR] → [POST-WEBINAR MAILS]
                                              ↓
                                        [SALES PAGE] → [CHECKOUT] → [ONBOARDING]
```

**Gebruik:** B2C coaches, kennisondernemers, therapeuten. Aanbod €500-€5.000+.

## Configuratie B: Product Launch

```
[AD] → [SALES PAGE] → [CHECKOUT] → [THANK YOU + ONBOARDING]
                            ↓
                      [ORDER BUMP optioneel]
```

**Gebruik:** Digitale producten, cursussen, boeken. Aanbod €27-€497.

## Configuratie C: Lead Magnet Funnel

```
[AD] → [REGISTRATIE PAGE] → [UPSELL] → [THANK YOU]
                                  ↓ nee
                            [DOWNSELL] → [THANK YOU]
                                  ↓ nee
                            [THANK YOU]

[NURTURE MAILS] → [SALES PAGE of WEBINAR INVITE]
```

**Gebruik:** Lijst opbouwen, cold traffic opwarmen. Lead magnet = gratis.

## Configuratie D: High-Ticket

```
[AD] → [REGISTRATIE/APPLICATIE PAGE] → [THANK YOU + INSTRUCTIES]
                                              ↓
                                   [QUALIFYING CALL] → [SALES CALL]
                                              ↓
                                        [SALES PAGE (optioneel)]
```

**Gebruik:** Programma's €3.000+. Persoonlijke verkoop vereist.

## Configuratie E: Simpel

```
[AD] → [REGISTRATIE PAGE] → [THANK YOU]
                                   ↓
                            [NURTURE MAILS]
```

**Gebruik:** Starters, simpele opt-ins, wachtlijsten.

## Routing Decision Tree

```
Prijs aanbod > €2.000?
├── Ja → Config D (High-Ticket)
└── Nee → Is er een webinar/masterclass?
    ├── Ja → Config A (Webinar Funnel)
    └── Nee → Is het een digitaal product?
        ├── Ja → Config B (Product Launch)
        └── Nee → Is het doel lijstopbouw?
            ├── Ja → Config C (Lead Magnet)
            └── Nee → Config E (Simpel)
```

## Congruence Checklist

Bij ELKE funnel, controleer:
- ☐ Ad belofte = registratiepagina headline
- ☐ Registratiepagina belofte = webinar/content inhoud
- ☐ Webinar inhoud = aanbod relevantie
- ☐ Sales page belofte = wat je daadwerkelijk levert
- ☐ Toon consistent door hele funnel (ToV-profiel)
- ☐ Visuele stijl consistent (kleuren, fonts, foto's)
