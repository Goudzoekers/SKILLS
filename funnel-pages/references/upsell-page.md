# Upsell Page — Direct Na Registratie

Verschijnt direct na de registratie. Doel: extra omzet + commitment verhogen (hogere show-up).

## Kernprincipe

De upsell moet VERSTERKEN wat ze net kochten/registreerden, niet afleiden. Het is een versneller, geen nieuw aanbod.

## Sectie-Opbouw

### 1. Bevestiging + Transition
```
[HEADLINE:] Gelukt! Je plek is gereserveerd.
[SUBTEXT:] Voordat je verdergaat — ik heb iets speciaals voor je.
```

### 2. Het Aanbod
```
[HEADLINE:] Wil je een voorsprong op [datum masterclass]?
[SUBHEADLINE:] [Naam upsell] — [kernbelofte in 1 zin]

Wat je krijgt:
✓ [Component 1] — zodat [verlangen] (waarde: €[X])
✓ [Component 2] — zonder [angst] (waarde: €[X])
✓ [Component 3] — zodat [verlangen] (waarde: €[X])

Totale waarde: €[HOOG]
Jouw investering: €[LAAG — typisch €27-€97]
```

### 3. Urgentie
```
Dit aanbod is alleen NU beschikbaar — op deze pagina.
Als je wegklikt, kun je het niet meer terugvinden.
```

### 4. CTA
```
[JA KNOP:] Ja! Geef mij de voorsprong → €[PRIJS]
[NEE LINK:] Nee bedankt, ik doe het zonder → (kleine tekst, onderaan)
```

## Pricing Sweet Spots

| Type upsell | Prijsrange | Voorbeeld |
|---|---|---|
| Template/checklist | €7-€27 | "De werkbladen bij de masterclass" |
| Mini-training | €27-€47 | "VIP voorbereiding: 3 video's vooraf" |
| VIP upgrade | €47-€97 | "Persoonlijke Q&A + bonus sessie" |
| Bundel | €97-€197 | "Complete toolkit + community toegang" |

## Regels

- NOOIT meer dan 1 upsell pagina na registratie (max 1 downsell als ze "nee" zeggen)
- De "nee" optie moet ALTIJD zichtbaar zijn — geen dark patterns
- Prijs moet <10% zijn van het hoofdaanbod
- Leverbaar VOOR het webinar (versterkt show-up)
- Betalingsfriction minimaal: 1-klik betaling als mogelijk
