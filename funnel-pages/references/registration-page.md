# Registration Page — Registratiepagina Template

Voor webinars, masterclasses, workshops, challenges. Primair doel: maximale registraties.

## Sectie-Opbouw

### 1. Hero (Above the Fold)
```
[HEADER:] Gratis [Format] voor [Avatar]
[HEADLINE:] [Resultaat] in [Tijdlijn] — zonder [Angst]
[SUBHEADLINE:] Ontdek hoe [avatar] [specifiek resultaat] bereikt
[DATUM + TIJD + DUUR]
[CTA KNOP:] Reserveer je gratis plek →
[SOCIAL PROOF:] "[X] mensen schreven zich al in"
```

### 2. Wat Je Leert (bullets)
```
In deze [masterclass] ontdek je:
✓ [Inzicht 1] — zodat je [verlangen]
✓ [Inzicht 2] — zonder [angst]
✓ [Inzicht 3] — zelfs als [bezwaar neutraliseren]
✓ [Inzicht 4] — zodat je [verlangen]
```
Max 5 bullets. Zonder/zodat-regel op elke bullet.

### 3. Voor Wie Is Dit
```
Deze [masterclass] is voor jou als:
✓ Je [herkenbare situatie 1]
✓ Je [herkenbare situatie 2]
✓ Je [herkenbare situatie 3]
```
Max 4 items. Gebruik "jij" taal.

### 4. Over de Spreker
```
[FOTO — professioneel, benaderbaar]
[Naam] — [1-zin kernresultaat]
• [Resultaat-bullet 1 — cijfer]
• [Resultaat-bullet 2 — cijfer]
• [Resultaat-bullet 3 — bekendheid/media]
```
Max 3 bullets. Resultaten, geen CV.

### 5. Social Proof
```
[3-5 testimonials of screenshots]
— of —
[Logos van bekende klanten/media]
— of —
"Sluit je aan bij [X]+ [avatar] die dit al ontdekten"
```

### 6. FAQ (optioneel, 3-4 vragen)
```
"Is het echt gratis?" — Ja, 100%. Geen creditcard nodig.
"Kan ik de replay bekijken?" — [Ja/Nee + reden]
"Hoeveel tijd kost het?" — [Duur] minuten.
"Voor wie is dit NIET?" — [Eerlijk antwoord]
```

### 7. CTA Herhaling
```
[DATUM + TIJD herhaling]
[CTA KNOP:] Claim je plek — het is gratis →
[SUBTEXT:] Beperkt aantal plekken. Geen creditcard nodig.
```

## Wireframe Notities

- Hero = 60% van de pagina-breedte op desktop, 100% op mobiel
- CTA knop: contrastkleur, groot, met witruimte eromheen
- Formulier: max 2-3 velden, inline of naast de hero
- Geen navigatiebalk — geen afleiding, geen exit-mogelijkheden
- Laadtijd: <3 seconden
- Responsive: test op iPhone, Android, tablet

## A/B Test Prioriteiten

1. **Headline** — grootste impact op conversie
2. **CTA tekst** — "Reserveer je plek" vs "Meld je gratis aan"
3. **Social proof** — met vs zonder testimonials
4. **Aantal formuliervelden** — 2 vs 3 velden
