# Sales Page — Long-Form Verkooppagina

Voor het hoofdaanbod. Doel: de prospect overtuigen om te kopen. Werkt samen met webinar of als standalone.

## Sectie-Opbouw (13 secties)

### 1. Hero
```
[HEADLINE:] [Resultaat] in [Tijdlijn] — zonder [Angst]
[SUBHEADLINE:] Het bewezen [systeem/methode] voor [avatar] die [specifieke situatie]
[CTA KNOP:] [Actie] — Start vandaag →
[SOCIAL PROOF:] "[X]+ [avatar] gingen je voor"
```

### 2. Probleem Herkenning
```
Herken je dit?
→ [Scenario 1 — specifiek, herkenbaar moment]
→ [Scenario 2]
→ [Scenario 3]

Als je bij minstens één ja knikte — lees dan verder.
```

### 3. Vijand Benoemen
```
Het is niet jouw schuld.
[Uitleg waarom de gangbare aanpak niet werkt]
[De echte oorzaak benoemen]
[Wat het je kost als dit zo doorgaat — concreet]
```

### 4. De Shift
```
Wat als [onverwacht inzicht]?
[Kort uitleggen waarom dit alles verandert]
[1-2 zinnen bewijs]
```

### 5. Introductie Aanbod
```
Daarom heb ik [NAAM PROGRAMMA] gebouwd.
[1-zin kernbelofte]
[Voor wie het is]
```

### 6. Wat Je Krijgt (modules)
```
Module 1: [Naam] — [Wat het oplevert] (waarde: €[X])
Module 2: [Naam] — [Wat het oplevert] (waarde: €[X])
Module 3: [Naam] — [Wat het oplevert] (waarde: €[X])
...
```
Per module: naam + resultaat + zonder/zodat. Max 7 modules.

### 7. Bonussen
```
Bonus 1: [Naam] — [Waarom relevant] (waarde: €[X])
Bonus 2: [Naam] — [Welk bezwaar het wegneemt] (waarde: €[X])
Bonus 3: [Naam] — [Specifieke waarde] (waarde: €[X])
```

### 8. Bewijs — Parade of Proof
```
[Case study 1 — naam, voor, na, tijdlijn]
[Case study 2]
[Case study 3]
[Screenshots / video testimonials]
```
Min. 5 vormen van bewijs. Mix diepte en breedte.

### 9. Prijs + Waarde Stack
```
Totale waarde: €[HOOG]
Jouw investering: €[PRIJS]
— of —
[X] termijnen van €[BEDRAG]

[CTA KNOP:] Start nu →
```

### 10. Garantie
```
[NAAM] Garantie:
Als je na [X weken/dagen] niet [specifiek resultaat] hebt bereikt,
krijg je [je geld terug / extra begeleiding / X].
[Voorwaarde als relevant]
```

### 11. Bezwaren (FAQ-stijl)
```
"Ik heb geen tijd" → [Antwoord]
"Het is te duur" → [Antwoord — ROI frame]
"Wat als het bij mij niet werkt?" → [Antwoord — bewijs + garantie]
"Ik heb al X geprobeerd" → [Antwoord — waarom dit anders is]
```

### 12. Crossroads Close
```
Je hebt twee opties:

Optie A: Niets veranderen → [concreet gevolg]
Optie B: Vandaag starten → [concreet resultaat]

De keuze is aan jou.
```

### 13. Finale CTA
```
[CTA KNOP:] [Actie] — Start vandaag →
[SUBTEXT:] [Garantie herhaling] | [Schaarste als relevant]
```

## Wireframe Notities

- Geen navigatiebalk — één doel, één pagina
- CTA knop min. 3x herhaald (na hero, na prijs, finale)
- Testimonials verspreid door de pagina, niet alleen in 1 blok
- Mobile-first: max 3 zinnen per alinea
- Contrast-kleuren voor CTA knoppen
- Witruimte tussen secties

## P.I.G. Integratie

- Hero = Primal Drijfveer (headline raakt de diepste pijn/verlangen)
- Sectie 2-3 = Problem verdieping (Primal)
- Sectie 4-5 = Identity Shift (van oud naar nieuw zelfbeeld)
- Sectie 6-9 = Growth Mechanism (het concrete middel)
- Sectie 12 = Crossroads Close (P.I.G. samengebracht)
