# Iteratie Protocol — Webinar-Specifiek

Optimalisatie-cyclus specifiek voor webinar funnels. Maandelijks doorlopen.

## Funnel Metrics & Benchmarks

| Stap | Metric | Benchmark | Meting |
|---|---|---|---|
| Ad → Klik | CTR | >1.5% | Ads Manager |
| Klik → Registratie | Opt-in CR | >25% | Landing page tool |
| Registratie → Show-up | Show rate | >30% | Webinar tool |
| Show-up → Einde | Retentie | >40% | Webinar tool |
| Einde → Klik aanbod | Pitch CR | >10% klik | Webinar tool |
| Klik → Aankoop | Close rate | >5-15% | Payment tool |

## Diagnose per Bottleneck

### CTR laag (<1.5%)
- **Oorzaak:** Hooks pakken niet, creative versleten, verkeerde doelgroep
- **Fix:** Nieuwe hooks → goated-ads. Creative refresh. Audience testen.

### Opt-in CR laag (<25%)
- **Oorzaak:** Headline matcht niet met ad, te veel velden, pagina te traag
- **Fix:** Headline A/B test → funnel-pages. Velden reduceren. Speed check.

### Show-up rate laag (<30%)
- **Oorzaak:** Te ver uit gepland, slechte reminders, geen incentive
- **Fix:** Max 72 uur vooruit plannen. Reminder-reeks optimaliseren. Incentive toevoegen.

### Retentie laag (<40%)
- **Oorzaak:** Opening pakt niet, te veel theorie, te weinig interactie
- **Fix:** Opening herschrijven. Meer polls/vragen. Kortere secties.

### Pitch CR laag (<10% klik)
- **Oorzaak:** Bridge te abrupt, aanbod onduidelijk, te weinig bewijs
- **Fix:** Bridge herschrijven. Aanbod visueel maken. Meer cases toevoegen.

### Close rate laag (<5%)
- **Oorzaak:** Prijs, bezwaren niet geadresseerd, sales page zwak
- **Fix:** Post-webinar mails. Sales page → funnel-pages. Bezwaar-mails.

## Waterval-Regel

Fix ALTIJD van boven naar beneden:
```
Ads → Opt-in page → Show-up → Webinar content → Pitch → Sales page → Post-mails
```
Het heeft geen zin om de pitch te optimaliseren als niemand komt opdagen.

## Maandelijkse Review Checklist

- ☐ Alle metrics verzameld per stap
- ☐ Grootste bottleneck geïdentificeerd
- ☐ Vergelijking met vorige maand
- ☐ 1-2 fixes geprioriteerd (niet 10)
- ☐ Fix geïmplementeerd
- ☐ Volgende meetmoment gepland (7-14 dagen)
