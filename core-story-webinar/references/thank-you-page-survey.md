# Thank You Page + Survey — Signaalverzameling

Na registratie. Twee doelen: bevestig de registratie en verzamel data voor personalisatie.

## Thank You Page Structuur

### Sectie 1: Bevestiging
```
[HEADLINE:] Je plek is gereserveerd!
[SUBHEADLINE:] Je ontvangt een bevestigingsmail op [e-mailadres]
[DETAILS:] [Datum] om [Tijd] — Check je inbox (en spam!)
```

### Sectie 2: Verwachtingen Zetten
```
Wat je kunt verwachten:
1. Je ontvangt nu een bevestigingsmail
2. 24 uur voor de masterclass krijg je een herinnering
3. Op de dag zelf ontvang je de link om in te loggen
```

### Sectie 3: Survey (zie onder)

### Sectie 4: Optionele Upsell
```
[HEADLINE:] Wil je een voorsprong?
[AANBOD:] [VIP upgrade / bonus / early access]
[PRIJS:] [Laag bedrag — €27-€97]
[CTA:] Ja, ik wil de voorsprong →
```

## Survey — 3-5 Vragen

De survey levert data die de hele funnel personaliseert: ads, mails, webinar-content, pitch.

### Verplichte Vragen

**Vraag 1: Situatie-check**
"Wat beschrijft jouw situatie het best?"
- [ ] Ik ben net begonnen met [niche-activiteit]
- [ ] Ik doe dit al een tijdje maar loop vast
- [ ] Ik heb al resultaten maar wil doorgroeien
- [ ] Anders: ___

**Vraag 2: #1 Uitdaging**
"Wat is op dit moment je GROOTSTE uitdaging met [onderwerp]?"
- [ ] [Optie A — meest voorkomend probleem]
- [ ] [Optie B]
- [ ] [Optie C]
- [ ] [Optie D]
- [ ] Anders: ___

**Vraag 3: Gewenst resultaat**
"Als deze masterclass je ÉÉN ding kon opleveren, wat zou dat zijn?"
[Open tekstveld]

### Optionele Vragen

**Vraag 4: Investering-bereidheid**
"Als er een manier was om [resultaat] te bereiken met begeleiding, wat zou je bereid zijn te investeren?"
- [ ] €0 — ik zoek gratis oplossingen
- [ ] €100-€500
- [ ] €500-€2.000
- [ ] €2.000+
- [ ] Hangt af van het resultaat

**Vraag 5: Tijdlijn**
"Wanneer wil je dit opgelost hebben?"
- [ ] Zo snel mogelijk
- [ ] Binnen 3 maanden
- [ ] Dit jaar
- [ ] Geen haast

## Data-Gebruik

| Survey-antwoord | Gebruikt in |
|---|---|
| Situatie-check | Segmentatie pre-mails + webinar voorbeelden |
| #1 Uitdaging | Webinar opening (poll data) + ad hooks |
| Gewenst resultaat | Webinar belofte + pitch framing |
| Investering-bereidheid | Lead scoring (kwalificatie) |
| Tijdlijn | Urgentie in follow-up mails |

## Technische Notities

- Survey moet op de thank you page zelf staan, NIET als aparte link
- Max 5 vragen — meer = drop-off
- Gebruik multiple choice waar mogelijk (makkelijker te segmenteren)
- Sla antwoorden op in CRM/tool gekoppeld aan het e-mailadres
- Completion rate benchmark: >60%
