# Opt-in Page — Webinar Registratiepagina

De eerste pagina die prospects zien. Doel: maximale registraties met gekwalificeerde leads.

## Sectie-Opbouw (van boven naar beneden)

### Sectie 1: Hero (Above the Fold)
```
[HEADER:] Gratis Masterclass / Training / Workshop
[HEADLINE:] [RESULTAAT] in [TIJDLIJN] — zonder [ANGST]
[SUBHEADLINE:] Hoe [AVATAR] [SPECIFIEK RESULTAAT] bereikt met [METHODE]
[DATUM + TIJD]
[CTA KNOP:] Claim je gratis plek →
```

**Regels:**
- Headline bevat het #1 gewenste resultaat
- Subheadline bevat avatar + methode
- Datum/tijd prominent zichtbaar
- CTA boven de vouw, geen scrollen nodig
- Nooit "gratis webinar" als combinatie — kies "gratis masterclass" of "gratis training"

### Sectie 2: Wat Je Leert (3-5 bullets)
```
In deze [masterclass/training] ontdek je:
✓ [Inzicht 1] — zodat [verlangen]
✓ [Inzicht 2] — zonder [angst]
✓ [Inzicht 3] — zelfs als [bezwaar]
✓ [Inzicht 4] — zodat [verlangen]
```

**Regels:**
- Max 5 bullets
- Elke bullet bevat zonder/zodat
- Motivatie, NIET activatie (wat ze LEREN, niet wat ze moeten DOEN)
- Specifiek: cijfers, tijdlijnen, namen waar mogelijk

### Sectie 3: Over de Spreker (kort)
```
[FOTO]
[NAAM] — [KERNRESULTAAT in 1 zin]
[2-3 credibility bullets: cijfers, niet CV]
```

**Regels:**
- Max 3 bullets
- Resultaten, geen kwalificaties
- Foto moet professioneel maar benaderbaar zijn

### Sectie 4: Social Proof
```
[TESTIMONIAL 1 — screenshot of quote]
[TESTIMONIAL 2]
[TESTIMONIAL 3]
— of —
"[X] mensen gingen je voor"
```

### Sectie 5: CTA Herhaling
```
[DATUM + TIJD — herhaling]
[CTA KNOP:] Reserveer je plek →
[SUBTEXT:] Gratis. Geen creditcard nodig. [X] plekken beschikbaar.
```

## Formulier-Velden

**Minimaal (hoogste conversie):**
- Voornaam
- E-mailadres

**Uitgebreid (betere kwalificatie):**
- Voornaam
- E-mailadres
- Telefoonnummer (optioneel)

**Regel:** Elk extra veld kost 5-15% conversie. Voeg alleen toe wat je ECHT nodig hebt.

## Mobile-First Checklist

- ☐ CTA zichtbaar zonder scrollen op mobiel
- ☐ Headline max 2 regels op mobiel
- ☐ Formulier max 2-3 velden
- ☐ Laadtijd <3 seconden
- ☐ Geen video die autoplay doet op mobiel
- ☐ Tekst leesbaar zonder zoomen
