# Partner Comms Templates — 5 Berichttypen

Gebruik als startpunt. Personaliseer altijd per klant.

## 1. Slack Onboarding

```
Hey [NAAM] 👋

Welkom bij Winstarchitect. Fijn dat je erbij bent.

Even kort hoe het werkt:

→ Ik ga eerst je briefing uitwerken (dat heb je al ingevuld)
→ Daarna krijg je een ToV-voorstel — dat moet kloppen voordat we verder gaan
→ Vervolgens bouw ik je Blueprint, en daarna de rest van je funnel

Wat ik van jou nodig heb: reageer op m'n berichten binnen 24 uur. Hoe sneller jij reageert, hoe sneller alles staat.

Vragen tussendoor? Stel ze hier. Ik reageer dezelfde werkdag.

Volgende stap: je ontvangt je ToV-voorstel binnen [X] werkdagen.

— Rick
```

## 2. Weekly Pulse

```
[NAAM] — Weekly Pulse [DATUM]

Focus deze week:
→ [Wat er gebouwd/geoptimaliseerd wordt]

Inzichten:
→ [1-2 observaties uit data of content]

Acties (door mij):
→ [Wat ik deze week oplever]

Input nodig van jou:
→ [Concrete vraag of actie — max 1]

— Rick
```

**Met campagnedata (datafeed uit analytics-feedback):**

```
[NAAM] — Weekly Pulse [DATUM]

Campagne update:
→ CPL deze week: €[X] (vorige week: €[Y])
→ Beste ad: [NAAM AD] — [WAAROM het werkt in 1 zin]
→ Actie: [schalen / nieuwe variaties / pagina tweaken]

Wat dit voor jou betekent: [vertaal naar klant-taal]

Focus komende week:
→ [Concrete actie]

— Rick
```

## 3. Milestone Update

**Variant A: Blueprint klaar**
```
[NAAM] — je Blueprint staat.

Ik heb je Perceptie Blueprint uitgewerkt. Dit is de architectuurtekening van je hele funnel: welke overtuigingen we installeren, in welke volgorde, en waarom.

→ [LINK NAAR DOCUMENT]

Lees het door en geef me feedback. Specifiek wil ik weten:
→ Herken je de beschrijving van je doelgroep?
→ Klopt de kernboodschap?

Zodra jij akkoord geeft, start ik met de funnel-onderdelen.

— Rick
```

**Variant B: Funnel review-ready**
```
[NAAM] — je funnel staat klaar voor review.

Alles is gebouwd:
→ [opsomming pagina's/ads/scripts]

Volgende stap: jij reviewt alles en geeft feedback. Ik heb per onderdeel een notitie toegevoegd waar ik specifiek jouw input nodig heb.

→ [LINK NAAR DOCUMENTEN]

— Rick
```

**Variant C: Live-ready**
```
[NAAM] — alles is klaar om live te gaan.

Je funnel heeft de Quality Gate doorlopen. Alles is gecheckt op copy, flow en techniek.

Volgende stap: jij zet het live (of geeft mij het signaal). Zodra de eerste data binnenkomt, stuur ik je een analyse.

— Rick
```

## 4. Performance Samenvatting

```
[NAAM] — Resultaten [PERIODE]

De korte versie:
→ [1 zin: gaat het goed, neutraal, of moet er iets veranderen?]

De cijfers:
→ Leads: [X] (vorige periode: [Y])
→ Kosten per lead: €[X]
→ Beste performer: [AD/PAGINA NAAM]
→ Aandachtspunt: [1 concreet ding]

Wat dit voor jou betekent:
→ [Vertaal naar hun taal: meer aanmeldingen, lagere kosten, etc.]

Wat ik ga doen:
→ [Concrete actie]

— Rick
```

## 5. Escalatie / Urgentie

```
[NAAM] — actie nodig

[Beschrijf in 1-2 zinnen wat er aan de hand is]

Wat ik van je nodig heb:
→ [Eén concrete actie]

Deadline: [WANNEER]

— Rick
```

## Template-regels

- Elk bericht past op één telefoonscherm
- Altijd afsluiten met "— Rick" (of naam uit Client Brief)
- Geen "Met vriendelijke groet"
- Geen verbindingstekens
- Maximaal 1 actievraag per bericht
- Klantnaam in elk bericht
- Geen jargon (CPL/CTR alleen als de klant het begrijpt, anders vertalen)
