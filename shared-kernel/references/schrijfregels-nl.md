# Schrijfregels NL — Mobile-First Copywriting

Laden bij ELKE Nederlandse copy-output. Geen uitzonderingen.

## Structuurregels

1. **Max 3 zinnen per alinea** — Op mobiel wordt meer dan 3 zinnen een muur
2. **Max 8 woorden per bullet** — Bullets zijn scan-elementen, geen paragrafen
3. **Geen verbindingstekens** — Geen "en/of", geen schuine streep combinaties
4. **Actieve vorm** — "Jij krijgt" niet "Er wordt aan jou geleverd"
5. **Groep 6 leesniveau** — Korte woorden, korte zinnen, geen bijzinnen in bijzinnen

## De Zonder/Zodat-Regel

Elk voordeel bevat twee delen:
- **Feature + "zonder [angst]"** — "Leer opvoeden zonder schreeuwen"
- **Feature + "zodat [verlangen]"** — "Leer opvoeden zodat je weer geniet van je gezin"

Nooit alleen de feature. Altijd de emotionele context erbij.

## Headline-Formules NL

```
[GETAL] manieren om [RESULTAAT] zonder [ANGST]
Hoe [AVATAR] [RESULTAAT] bereikt in [TIJDLIJN]
Stop met [FOUTE AANPAK]. Dit werkt wél.
Waarom [VEELGEBRUIKTE METHODE] niet werkt (en wat wel)
De [GETAL] fouten die [AVATAR] maakt bij [ONDERWERP]
```

## CTA-Formules NL

```
Klik hieronder en [ACTIE]
Meld je nu aan (gratis)
Claim je plek — [SCHAARSTE]
Start vandaag — [RESULTAAT-BELOFTE]
Bekijk de gratis training →
```

**Regel:** CTA = instructie. Vertel exact wat ze moeten doen. Minimaal 3x per pagina herhalen.

## Woorden die NIET mogen

| Verboden | Vervang door |
|---|---|
| Transformatie | Verandering, shift, omslag |
| Ontgrendel | Ontdek, leer, krijg toegang tot |
| Geheim | Inzicht, aanpak, methode |
| Magische formule | Bewezen systeem, werkwijze |
| Reis/journey | Traject, pad, weg |
| Harmonie | Rust, balans, ruimte |
| Vrijheid (abstract) | Concreet: "zelf bepalen wanneer je werkt" |
| Uniek | Bewezen, specifiek, anders dan |

## Interpunctie & Opmaak

- **Vetgedrukt** = alleen kernwoorden, max 2 per alinea
- *Cursief* = nadruk, spaarzaam
- HOOFDLETTERS = alleen in headlines of extreme nadruk, max 1 woord
- Emoji's = max 1 per bericht, alleen in ads/social, nooit in e-mails
- Drie puntjes... = spanning opbouwen, max 1x per tekst

## Scan-Patroon Optimalisatie

Mensen lezen in F-patroon:
1. Eerste 2 woorden van elke regel worden ALTIJD gelezen → zet daar de kernboodschap
2. Headlines worden gelezen → maak ze zelfstandig begrijpelijk
3. Bullets worden gelezen → maak ze compleet zonder context
4. Eerste en laatste zin per sectie worden gelezen → power positions

## Lengte-Richtlijnen

| Type | Lengte |
|---|---|
| Ad hook | 5-15 woorden |
| Ad body | 4-6 zinnen |
| E-mail onderwerp | 4-8 woorden |
| E-mail body | 150-300 woorden |
| Landing page headline | 8-15 woorden |
| Bullet point | Max 8 woorden |
| Sales page | 1500-3000 woorden |
| VSL script | 800-2000 woorden |
